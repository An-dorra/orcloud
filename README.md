# OrCloud Vue Starter

Vue 3 + TypeScript + Tailwind CSS + Vue Router starter project.

## Scripts

```bash
npm run dev
npm test -- --run
npm run typecheck
npm run build
```

## Structure

- `src/components/base/`: reusable base UI components
- `src/components/layout/`: shared layout components
- `src/components/feature/`: reusable feature components
- `src/data/`: typed shared data
- `src/router/`: Vue Router setup and route tests
- `src/views/`: route-level page components
